This repository features a real-time drowsiness detection system that utilizes a dual tracking pipeline: an optimized local inference app powered by a custom MobileNetV2 binary eye classifier and traditional geometric Dlib Eye Aspect Ratio (EAR) calculations. The production backend is built on a thread-safe, rate-limited Flask framework engineered to process streaming webcam frames via asynchronous base64 decoding with active debugging. The machine learning pipeline integrates mixed-precision training, custom multi-task localization loss, and data augmentation to deliver highly responsive, low-latency driver safety monitoring. It serves as a robust framework for edge-sensing computer vision deployment, balancing deep learning accuracy with lightweight hardware execution.
